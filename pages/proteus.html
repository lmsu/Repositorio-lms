TEMARIO CURSO DE PROTEUS


  • Módulo 1: Introducción al Proteus.
    1. Requisitos necesarios para la instalación.
    2. Pantallas, Configuración y herramientas del programa
    3. Librerías de componentes.
    4. Instrumentos de medición
    5. Medición de las diferentes magnitudes eléctricas
    6. Análisis de circuitos electrónicos más usados.

  • Módulo 2: Creación de diagramas esquemáticos
    1. Cómo iniciar la creación de un nuevo circuito.
    2. Diagrama esquemático, búsqueda de componentes
      en las librerías.
    3. Montaje y conexión de los mismos.
    4. Cómo cambiar nombres de componentes
      y agregar etiquetas.
    5. Análisis de esquemas de circuitos electrónicos.

  • Módulo 3: Librerías de componentes.
    1. Creación de librerías y componentes esquemáticos
      que no se encuentran en las librerías.
    2. Definición de footprints.
    3. Cómo crear librerías y footprints.
    4. Cómo asociar un footprint con un símbolo esquemático.
    5. Ejemplos prácticos.

  • Módulo 4: Simulación de circuitos.
    1. Simulación, consideraciones a tener en cuenta.
    2. Cómo usar el osciloscopio, voltímetro, amperímetro,
      frecuencímetro, vatímetro y demás instrumentos virtuales.
    3. Interpretación de gráficos y resultados.
    4. Ejemplos de simulaciones de circuitos.

  • Módulo 5: PCB Layout.
    1. Cómo definir el tamaño de nuestro circuito impreso.
    2. Capas, pistas y plano de masa del circuito impreso.
    3. Diseño del circuito impreso de manera automática
      y manual.
    4. Cambio de grosor de las pistas.

  • Módulo 6: Visualización 3D.
    1. Visualización 3D del circuito terminado.
    2. Cómo crear un modelo 3D de un componente que no
      cuenta con el mismo.
    3. Uso de la paleta de colores.
    4. Creación de componentes propios.

  • Módulo 7: Archivos de Salida.
    1. Generación de archivo Gerber para pedidos a China.
    2. Cómo hacer el pedido a PCBway, pago y seguimiento
      del envío.
    3. Fabricación casera de Circuitos Impresos.
    4. Impresión de PDF para método de estampado.
    5. Fabricación casera de un PCB. Ejemplo en vivo.

  • Módulo 4: Simulación de circuitos.
    1. Simulación, consideraciones a tener en cuenta.
    2. Cómo usar el osciloscopio, voltímetro, amperímetro,
      frecuencímetro, vatímetro
      y demás instrumentos virtuales.
    3. Interpretación de gráficos y resultados.
    4. Ejemplos de simulaciones de circuitos.
